# ISDLmap
研究室内の部屋の使用状況を一目で把握できるWebアプリケーション
Google Calendar APIを用いて研究室の教員および学生の予定を取得し，研究室内の様子を一目で！

# 使用技術
- フロントエンド：HTML，CSS，JavaScript
- バックエンド：Flask
- データベース：MySQL
